# Prediction-of-Product-Sales
## Project Overview
This project aims to analyze product sales data and identify the key factors that influence sales performance. Through Exploratory Data Analysis (EDA), we uncover patterns and relationships between different features such as item type, outlet characteristics, and pricing.
The goal is to better understand what drives higher sales and support data-driven decision-making.
